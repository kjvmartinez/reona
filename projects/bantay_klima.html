Bantay Klima

Establishing a Hybrid Weather Monitoring Network Across External Campuses of Palawan State University

I. Implementing HEI

Palawan State University (PalawanSU)
College of Sciences
Puerto Princesa City, Palawan

II. Project Proponent

Prof. Ronald Edilberto A. Ona, PhD
Dean, College of Sciences

III. Project Duration

24 Months

IV. Rationale

Palawan is increasingly exposed to climate variability, including typhoons, erratic rainfall, and droughts. Current weather monitoring systems lack localized resolution needed for decision-making, research, and community resilience.

Bantay Klima proposes a hybrid monitoring network combining commercial weather stations and DIY IoT-based sensors to generate real-time, localized climate data across Palawan State University’s external campuses.

V. Objectives

  • Deploy commercial weather stations in selected campuses
  • Install IoT-based smart weather sensors
  • Develop a centralized data platform
  • Generate localized climate datasets
  • Enhance institutional capacity in IoT and analytics
  • Support climate adaptation and disaster preparedness

VI. Project Components

1. Commercial Weather Stations

  • High-accuracy, calibrated systems
  • Measure temperature, humidity, rainfall, wind, solar radiation
  • Serve as reference stations

2. DIY IoT Smart Sensors

  • ESP32 / Raspberry Pi platforms
  • Low-cost sensor integration
  • Solar-powered with wireless connectivity

3. Data Management System

  • Cloud/local server infrastructure
  • Real-time dashboard
  • Database for historical analysis

4. Capacity Building

  • Training for faculty and students
  • Integration into academic programs

VII. Implementation Plan

Phase Activities
Phase 1 Planning, site selection, procurement
Phase 2 Installation of weather stations and sensors
Phase 3 System integration and dashboard development
Phase 4 Testing and calibration
Phase 5 Training and academic integration
Phase 6 Monitoring and evaluation

VIII. Expected Outputs

  • Installed hybrid weather monitoring systems
  • Operational real-time data platform
  • Localized weather datasets
  • Technical documentation

IX. Expected Outcomes

  • Improved climate data availability in Palawan
  • Enhanced research and analytics capability
  • Strengthened disaster preparedness
  • Increased student technical competencies

X. Sustainability Plan

  • Integration into Environmental Analytics Laboratory
  • Partnerships with government and research institutions
  • Continuous funding through grants and programs

XI. Risk Management

Risk Mitigation
Equipment damage Protective enclosures and proper siting
Connectivity issues Hybrid communication systems
Maintenance constraints Training local personnel
Data inconsistency Calibration with reference stations

XII. Conclusion

Bantay Klima positions Palawan State University as a leader in environmental monitoring and climate resilience through a scalable, hybrid system integrating commercial and IoT technologies.
